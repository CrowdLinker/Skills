# Supported Agents

These skills follow the [Agent Skills open standard](https://agentskills.io) and are compatible with any agent that supports `SKILL.md` files.

| Agent | Install method |
|-------|---------------|
| [Claude Code](https://claude.ai/code) | `npx skills add crowdlinker/skills` |
| [Codex CLI](https://github.com/openai/codex) | `npx skills add crowdlinker/skills` |
| [Gemini CLI](https://github.com/google-gemini/gemini-cli) | `npx skills add crowdlinker/skills` |
| [Cursor](https://cursor.com) | Copy `SKILL.md` content into `.cursorrules` or system prompt |
| [GitHub Copilot](https://github.com/features/copilot) | Copy `SKILL.md` content into custom instructions |
| [Cline](https://github.com/cline/cline) | `npx skills add crowdlinker/skills` |
| [Windsurf](https://windsurf.ai) | Copy `SKILL.md` content into global rules |
| Any MCP-compatible agent | Copy `SKILL.md` content into system prompt |
